% \newcommand-定义命令
% \newcommand<命令名称>[<参数个数>][<首参数默认值>]{<具体定义>}
\newcommand\PRC{People's Republic of \emph{China}}
% 这个命令就是一个宏
% 而宏就是替换，只是你可以规定你替换什么内容罢了，是全部替换还是设置参数进行部分替换

% 如果指定了参数的个数 (从 1 到 9 ), 则可以在宏的定义中使用参数。在〈具体定义〉中, 
% 宏的参数依次编号, 用 #1, #2,..., #9 表示, 使用宏时它们【#1， #2等】会替换为参数的字符串, 例如:
% 参数个数可以从1到9，使用时用#1，#2，#3，#4...#9
\newcommand\loves[2]{#1 喜欢 #2}
% 例如： \loves{猫}{鱼}；会把#1替换成猫，把#2替换成鱼。所以这句话就是“猫喜欢鱼”

% newcommand的参数也可以有默认值
%指定参数个数的同时指定了首个参数的默认值，那么这个命令的第一个参数就可以成为可选参数（要使用中括号指定）
\newcommand\lovemore[3][喜欢]{#2#3#1}
% 首个参数就是#2，所以#2默认为喜欢
% 可以用新的可选参数覆盖原来的默认内容
